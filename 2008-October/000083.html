High Leute,

ich habe in letzter Zeit mich mal wieder ein bischen mehr mit der
Zsh aus einander gesetzt. Vieles muss ich aus den kwtools noch
entfernen, wie "setopt shwordsplit". Das das nicht gerade einfach
ist wird das ein schleppender Prozess sein. A'la alles kennzeichnen
mit "unsetopt shwordsplit". Die Scripte brauchen halt für "dialog"
noch diese Option. Danach mit einem Rutsch alles entfernen.

In allen Scripts wurde der Header verbessert und die meisten haben
jetzt auch eine Überprüfung ob wichtige fehlende Programme fehlen.
Also nicht warten auf die Überprüfung bei der Ausführung.

Ich mache noch ein paar Tests mit kwpostfix, kwlosetup und
kwcryptsetup. Ansonsten läuft bisher alles.

Die html Doku ist aktuell, ohne offizielle Anmerkungen zu Gentoo. Es
fehlt halt noch der Support für einige Scripte, wie kwnetcardconf.

cvs ist aktuell.

bisheriger changelog:
kwtools (0.7.3-1) unstable; urgency=low
  * kwparted: Status display/Help bug fixed. The calculation of
    partition size to changed (KByte -> Byte).
  * kwquota: Support for "JFS" filesystem. Support for warnquota
    to configure.
  * only support udev (filesystems).
  * kwwvdialconf: Support for tel-nr (*99# - Handy data).
  * to calculate the drive size was bettered.
  * kwnetcardconf: Support for WLAN Configuration.
  * Function convert_drive_size: calculation of *Byte bettered.
  * Function os_check update (Fedora -> fedora, cat /proc/version)
  * Scripts update to os_check.
  * new function check_part_fs (filesystemtyp) and convert_si_size
    (1MByte = 1000.000 Byte).
  * kwvmstat: new - no part to display - Info message.
  * kwgrub: delete Bug fixed.
  * kwnarc: initscript update.
  * main.cf and functions kwadmin_config - support for color update.
  * kwrunlevconf: Support for openrc (gentoo).
  * kwpostfix: *delimiter* to exist are earmarked for dialogbox.

kwtools (0.7.3-1) unstable; urgency=low
