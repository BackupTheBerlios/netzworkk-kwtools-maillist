High Leute,

habe am WE die neue Release fertig gestellt.

Neue Variablendefinitionen kamen nicht hinzu. Es wurden einige
Scripts umbenannt (name --> kwname), vor allem in den Bereichen
multi und net. Einige Scripts hatten Bugs in der Anzeige von
Fehlern. So wurde zum Beispiel nicht "Hallo, foo ist nicht da."
angezeigt, sondern "Hallo, Achtung ist nicht da.".

changelog:
kwtools (0.6.0-5) unstable; urgency=low
  * the functions resize_fs and mount_check for Debian/sid updated.
  * Script termin_cron Bug fixed, not removed the directory in ~/tmp
  * update kwrsync_backup (help, config, man5, html)
  * rename djbdns and proxy to kwdjbdns and kwproxy (manpage too).
  * update kwdjbdns (Attention display, help, html)

Nachdem durchsehen von kwdjbdns werde ich die Funktion ccde_conf
dort einfuehren (ccde = create/change/delete/exit).

Die Doku-Seiten wurden im Bereich "net" ausfuehrlich erweitert (z.B.
kwdjbdns).

--------------------------------------------------------------------
Achtung: Wer die Sourcen benutzt sollte immer zuerst das uninstall
Script der alten Release benutzen, da das neue uninstall Script nat.
an die neuen Verhaeltnisse angepasst ist.
--------------------------------------------------------------------
