High Leute,

gegenueber der 0.6.0-0 er Release hat sich ne Menge getan.
Vor allem im Multimedia Bereich. So kann kwrecord neue Funktionen
zum brennen, snd2sndfm die en(de)coder Optionen einstellen.
Die Funktion cd_dvd_burn kann jetzt Dateien brennen die mehr als
2 Leerzeichen in Folge besitzen. Lag wie schon mal behauptet an
mkisofs. kwrecord kopiert nun mit und ohne temporaeren Verz. CDs und
DVDs. Beim CD kopieren kann man eine CDDB abfragen.
kwgrub unterstuetzt nun die Parameter "hiddenmenu" und "splashimage".

changelog:
kwtools (0.6.0-1) unstable; urgency=low
  * function kwrsync_backup_exec - display for mails (cron - subject) updated.
  * function script_backtitle - year updated.
  * functions if_conf_file, kwgrub_conf for grub extended.
    (hiddenmenu and splashimage).
  * manpages from kwgrub.8, kwrecord.1, snd2sndfmrc.5, snd2sndfm.1
    and cd_dvdrc.5 updated.
  * function cd_dvd_burn add CDDB Support (only Option --cdcopy),
    on-the-fly and tmp directory.
  * function cd_driver supported flags for cdrdao driver.
  * function kwrecord_conf supported configuration from a tmp directory,
    CDDB and the Option on-the-fly.
  * snd2sndfm can configure the decode and encode options.
  * functions mount_check and resize_fs added readlink for devices.
  * docs, help, updated.
  * function file_filter for MP3 updated.
  * Bugs fixed and BUGS file updated (cd_dvd_burn+mkisofs).

Es wurde angefangen die Scripte zu vereinheitlichen. So gibt es
bereits einige definierte Variablen. Bis auf das die Variablen in
den Scripten/Funktionen existieren gibts noch keine Doku dazu.

Variablendeklaration der kwtools:

---------------------------------------------------
Globale Variablen in jeglicher Hinsicht:

gv_ScriptName - Name des Scripts

gv_Auswahl - Rueckgabewerte von dialogs Menueboxen
gv_ChangeValue - yes/no
gv_CreateValue - yes/no
gv_DeleteValue - yes/no

Uebersetzungs-Variablen:
gv_Attention - Achtung/Attention
gv_Back - abbrechen/back
gv_Change - aendern/change
gv_Configuration - Konfiguration/Configuration
gv_Create - anlegen/create
gv_Delete - loeschen/delete
gv_Exit - beenden/exit
gv_Info - Info
gv_Mainmenu - Hauptmenue/Mainmenu
gv_Save - speichern/save

gv_Menupoint - Waehlen Sie bitten einen Menuepunkt aus.

---------------------------------------------------
Lokale Variablen:

---------------------------------------------------
Globale Variablen in Funktionen:
*TITLE - Titel der dialogbox (Bsp.: TITLE/TYP_TITLE)
*MSG - Nachricht der dialogbox (Bsp.: MSG/TYP_MSG)
*MENU - Werte die die dialogbox zur Auswahl anzeigen soll
(Bsp.: MENU/TYP_MENU)

---------------------------------------------------
Lokale Variablen in Funktionen:
Bsp. Schreibweise: Anz, Tmp, ...

mehr ist noch nicht.
