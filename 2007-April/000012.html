High Leute,

Release kwtools-0.6.0-3 ist fertig.

Alle Multimedia Scripts haben jetzt das Prefix kw. Die Scripts
kwcamera, kwrip, kwfilm2film, kwsnd2sndfm, kwvcdburn und kwvdvdburn
besitzen eine Funktion um die alte Konfiguration zu uebernehmen.
Desweiteren wurden die Paketabhaengigkeiten beim Paket
kwtools-utils-pim geaendert. Es wurde tetex* mit texlive* (nur Debian)
getauscht und getestet.

changelog:
kwtools (0.6.0-3) unstable; urgency=low
  * function cd_driver updated.
  * Script kwrip updated (full CD rip).
  * Scripts vcdburn, vdvdburn, snd2sndfm rename to kw${Old_Name}.
  * function for update from configurations files from scripts to rename.
  * kwtools-utils-pim update from package tetex to package texlive. 


Bis auf die Pakete (in Arbeit) wird wohl Morgen alles uptodate
sein (cron).
