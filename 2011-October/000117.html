Nummer 0.8.4. Sollte (auch laut website) erst Nummer 0.8.3-4 werden.

einfliessen und auch getestet.

------------------------------cut-----------------------------
neues Script kwssl2 (kwssl -> kwssl2):
- RootCA erstellen
- SubCA erstellen
- Server Zertifikate erstellen
- Benutzer Zertifikate erstellen
- Konfiguration von Benutzer Zertifikaten, Server Zertifikaten, CA und
  der Gueltigkeit in Tagen von CRLs.
- exportieren
- ungueltig erklaeren
- loeschen

Importieren von CAs oder Zertifikaten geht noch nicht.
Auch gibts es noch ein Problem bei der Auswahl des Algorithum (DSA oder
RSA) bei der Erstellung von CAs oder Zertifikaten. Wird naechste Woche
bereinigt (heute Geburtstagsfeier).

Desweiteren liegt das Script kwssl2 jetzt im Verzeichnis ../utils und
benoetigt keine Root Rechte mehr.
