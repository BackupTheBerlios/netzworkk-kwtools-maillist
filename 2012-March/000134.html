Plugin System:
In der Variable PLUGINS (~/.zshrc.d/020-variables.z) kann man angeben
welche Plugins beim Start der "zsh" ausgefÃ¼hrt werden sollen.

Dies sind zsh-Scripte die beim Start der "zsh" mittels "source"
eingelesen und ausgefÃ¼hrt werden. Dabei kann eine Tastatur Eingabe
erforderlich sein.

Zur Zeit existieren nur die Plugins autojump, dirpersist, gpg-agent
und ssh-agent. Alle Plugins liegen unter ~/.zplugins und haben die
Endung .plugin.

autojump   - siehe dem Debian Paket autojump
dirpersist - speichert beim ausloggen die Ausgabe von "dirs -p | uniq"
             in der Datei ~/.zdirs und erstellt beim einloggen aus
             dieser Datei den $dirstack von "dirs".
gpg-agent  - startet den gpg-agent.
ssh-agent  - startet den ssh-agent. Siehe auch den Header der Datei.
