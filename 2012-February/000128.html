Moin,

so die neue Release ist fertig. Dabei wurde nicht nur das Script kwssl2
fertig erstellt und kwquota weiter entwickelt, sondern auch der Paketbau
upgedatet.

----------------------------cut---------------------------------
Paketbau:
* benutzt jetzt das debhelper Programm dh_lintian und dadurch
  Umbenennung der debian/*.overrides.lintian ->
  debian/*.lintian-overrides
* Pakete die keine Extra Dokumentation besitzen sind jetzt ein Link auf
  das Dokumentationsverzeichnis vom Paket kwtools-common.
* Es gibt nur noch eine englische changelog Datei im
  Dokumentationsverzeichnis, da changelog und changelog.Debian identisch
  sind.
* die Release Dateien im Repo-Baum wurden verbessert.

Notiz: Die Version der Release Dateien, im Repository, ist diesmal noch
0.8.4, also gleiche Versionsnr. wie beim letzten erstellen des
Repository. Da war leider alles schon hoch geladen.

----------------------------cut---------------------------------
changelog:
* kwssl2: Neu - Support fuer Schluessel, Zertifikat und Request
  (Anforderung) Management. Import einer CA + deren Zertifikate.
* kwquota: mehr locale Unterstuetzung.

----------------------------cut---------------------------------
ftp Download:
ftp ftp://ftp.berlios.de/pub/netzworkk/scripts/kwtools/upload/stable

Debian Pakete - /etc/apt/sources.lst:
deb ftp://ftp.berlios.de/pub/netzworkk/scripts/debian-kwtools stable main
deb-src ftp://ftp.berlios.de/pub/netzworkk/scripts/debian-kwtools stable main

So viel Spass beim ausprobieren.
