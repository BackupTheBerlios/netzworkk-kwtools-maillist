Hallo zusammen,

kwparted erreicht langsam wieder stable Status. Alle Tests liefen
bisher nur unter grml-1.1 (Live-CD, Debian/i386).

Was geht:
- anlegen von Partitionstabellen, nur Label msdos, wenn keine
  existiert.
- anlegen und löschen von Partitionen, mit den Dateisystemen ext2/3,
  vfat, hfs(+), jfs, ntfs, reiserfs, reiser4, ufs und xfs.
- vergrössern/verkleinern von Partitionen ohne Dateisystem.
- vergrössern/verkleinern von Partitionen mit Dateisystem, bisher
  nur ext2.
- bei bestehenden Partitionen kann man das Flag (lvm, raid,...)
  ändern. Scheint aber nicht richtig unter unstable Debian im Batch
  Modus zu funktionieren. parted ... set boot on, raid on -> führt
  dazu das nur das boot Flag gesetzt wird, raid nicht.
- Zur Berechnung der Laufwerks Grössen wird als erstes die Grösse der
  Zylinder in Bytes (DISK_CYL_BYTES) heraus gefunden und zur
  Berechnung benutzt.
- Die Funktion disk wurde erweitert um die Variable DISK_CYL_BYTES.

Das kopieren von Partitionen mit/ohne Dateisystem wurde noch nicht
getestet.

Hilfe wurde erweitert.

Alles aktuell im cvs.
